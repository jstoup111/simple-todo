# CLAUDE.md — Todo API

## Project

Simple todo REST API. No authentication, no users. Todos can be created and marked as done.

- **Framework:** Rails 7+ (API mode)
- **Database:** PostgreSQL
- **Test framework:** RSpec
- **Frontend:** API-only (no views)

## Key Commands

```bash
# Start server
rails server

# Run all tests
bundle exec rspec

# Run a single spec
bundle exec rspec spec/path/to/file_spec.rb

# Database
rails db:create db:migrate
rails db:rollback           # undo last migration

# Rails console
rails console
```

## Conventions

- Controllers in `app/controllers/api/` namespaced under `Api::`
- JSON responses via `render json:`
- RSpec request specs for API endpoints (`spec/requests/`)
- Factories via FactoryBot (`spec/factories/`)
- Thin controllers — business logic in model or service if needed

## Worktree Compatibility

Database name is parameterised: `todo_#{ENV.fetch('WORKTREE_DB_SUFFIX', Rails.env)}`.
Set `WORKTREE_DB_SUFFIX` when running parallel worktrees to avoid DB collisions.
