# Implementation Plan: Todo API

**Date:** 2026-04-02
**Design:** `docs/specs/2026-04-01-todo-api.md`
**Stories:** `docs/stories/todo-api.md`
**Conflict check:** Skipped (S-tier complexity, single-feature, no conflicts possible)

## Summary

Builds a Rails API-only todo service with two endpoints: `POST /api/todos` (create) and
`PATCH /api/todos/:id/complete` (mark done). 10 tasks covering all happy and negative paths.

## Prerequisites

- Rails app already bootstrapped in API mode with PostgreSQL configured
- RSpec + FactoryBot gems present in Gemfile
- `rails db:create` has been run
